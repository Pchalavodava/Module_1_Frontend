Основы HTML

Задание:

Задание: создать собственную HTML-страницу.

Начните с определения базовой структуры, используя теги html, head, title и body. Добавьте несколько текстовых элементов и заголовков на страницу.

ВАЖНО.

Новый файл создаем, используя VS Code или WebStorm. Если попытаться создать его без этих программ, далее (IDE), то на Windows будет создан файл вида index.html.txt, если у вас скрыты расширения файлов Поэтому, мы работаем сразу, используя IDE! Прикрепляем не просто файл, а создаем архив. В архив помещаем все файлы, которые использовали :)

Важно 2.

В названии файлов используем ТОЛЬКО английские буквы
